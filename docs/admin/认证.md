# 登录

#### 简要描述：

- 后台登录接口

#### 请求接口:

- /api/admin/login

#### 请求方式：

- POST

#### 请求参数:

|参数名|是否必须|类型|说明|
|:----    |:---|:----- |-----   |
|username |是  |string |登录用户名 |
|password |是  |string | 密码    |
